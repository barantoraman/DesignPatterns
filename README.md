# GoF Design Patterns in Go

This repository contains implementations of **Gang of Four (GoF) Design Patterns** in **Go**. 


##  Patterns Covered
The patterns are grouped into three categories: **Creational**, **Structural**, **Behavioral**

### Creational
- [X] Singleton
- [X] Factory
- [X] Abstract Factory
- [X] Builder

### Structural
- [X] Adapter
- [X] Bridge
- [X] Decorator
- [X] Facade
- [X] Proxy
